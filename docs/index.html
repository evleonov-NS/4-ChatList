Версия 1.1.0 · Windows

ChatList

Отправьте один промт в несколько нейросетей, сравните ответы рядом и сохраните лучшие результаты в SQLite.

Скачать установщик GitHub Документация

Возможности

Один промт — много моделей

Параллельная отправка в активные модели OpenRouter и сравнение ответов в таблице.

AI-ассистент

Улучшение промтов перед отправкой с помощью нейросети.

История и экспорт

Сохранение в SQLite, экспорт в Markdown и JSON, журнал запросов.

Бесплатные модели

Gemma, GPT-OSS и OpenRouter Free — без пополнения баланса для тарифа :free.

Установка

  1. Скачайте ChatList-Setup-1.1.0.exe со страницы Releases.
  2. Запустите установщик и завершите мастер установки.
  3. В каталоге программы скопируйте .env.example в .env.
  4. Укажите ключ OpenRouter (получить ключ):
OPENROUTER_API_KEY=sk-or-v1-...
OPENAI_BASE_URL=https://openrouter.ai/api/v1
  1. Запустите ChatList из меню «Пуск».

Для разработчиков

Python 3.11+, PyQt6, SQLite, httpx.

git clone https://github.com/evleonov-NS/4-ChatList.git
cd 4-ChatList
python -m venv .venv
.\.venv\Scripts\Activate.ps1
pip install -r requirements.txt
Copy-Item .env.example .env
python main.py

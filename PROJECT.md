# Спецификация программы ChatList

## 1. Назначение программы

ChatList — Python-приложение для отправки одного промта в несколько нейросетей и сравнения ответов. Отмеченные строки сохраняются в SQLite.

## 2. Стек

- Python 3.11+
- SQLite · httpx · PyQt6 · python-dotenv · Pillow (генерация иконки)

## 3. Таблицы базы данных

| Таблица | Назначение |
|---------|------------|
| `prompts` | Сохранённые запросы |
| `models` | Нейросети и параметры API |
| `results` | Сохранённые ответы |
| `settings` | Настройки программы (`theme`, `font_size`, …) |

Подробнее: [DATABASE.md](DATABASE.md).

## 4. Рабочий процесс

1. Ввод или выбор промта, опционально **улучшение через AI-ассистент**.
2. Параллельная отправка в активные модели OpenRouter.
3. Сравнение ответов во временной таблице, отметка галочками.
4. Сохранение выбранных строк в `results`.

## 6. Настройки (этап 13)

**Меню:** Настройки → Настройки…

| Параметр | Ключ в `settings` | Описание |
|----------|-------------------|----------|
| Тема | `theme` | `light` или `dark` |
| Размер шрифта | `font_size` | 8–20 pt, панели главного окна |

При сохранении вызывается `themes.apply_appearance()`.

Расширенный редактор всех пар key/value: **Настройки → Расширенные параметры…**

## 7. О программе (этап 14)

**Меню:** Справка → О программе…

Диалог `AboutDialog`: название, версия, описание, список возможностей, стек технологий.

## 8. AI-ассистент улучшения промтов

Кнопка **«Улучшить промт»** → одна модель → улучшенный текст, альтернативы, адаптации → подстановка в поле ввода.
